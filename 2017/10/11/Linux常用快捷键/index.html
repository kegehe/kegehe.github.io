Linux常用快捷键

发表于 2017-10-11
字数统计: 759 | 阅读时长 ≈ 3

 在Linux系统中，我们通常会在系统上进行大量的命令和操作，所以掌握一定的Linux快捷键用法会让我们的工作更高效，这也是一种由新手变高手的蜕变过程。

删除

快捷键 说明
Ctrl + d 删除光标所在位置上的一个字符(若当前没有任何，退出shell)
Ctrl + h 删除光标所在位置前的一个字符
Ctrl + k 删除光标后面所有字符(包括自身)
Ctrl + u 删除光标前面所有字符(不包括自身)
Ctrl + w 删除光标前一个单词
Ctrl + ? 撤消前一次输入
Alt + r 撤消前一次动作
Alt + d 删除光标所在位置的后单词

移动

快捷键 说明
Ctrl + a 将光标移动到命令行开头
Ctrl + e 将光标移动到命令行结尾处
Ctrl + f 光标向前移动一个字符,相当与→
Ctrl + b 光标向后移动一个字符,相当与←
Ctrl + ← 光标移动到前一个单词开头
Ctrl + → 光标移动到后一个单词结尾
Ctrl + x 在上次光标所在字符和当前光标所在字符之间跳转
Alt + f 光标向前移动一个单词
Alt + b 光标向后移动一个单词
Esc + b 移动到当前单词的开头
Esc + f 移动到当前单词的结尾

替换

快捷键 说明
Ctrl + t 将光标当前字符与前面一个字符替换
Ctrl + y 粘贴刚才所删除类删除的字符
Ctrl + v 插入特殊字符,如Ctrl + v + Tab加入Tab字符键而不是扩展命令行
Alt + t 交换两个光标当前所处位置单词和光标前一个单词
Alt + u 把光标当前位置单词变为大写
Alt + l 把光标当前位置单词变为小写
Alt + c 把光标当前位置单词头一个字母变为大写
Esc + t 颠倒光标所在处及其相邻单词的位置

历史命令编辑

快捷键 说明
↑/(Ctrl + p) 显示上一条命令
↓/(Ctrl + n) 显示下一条命令
Ctrl + r 输入单词搜索历史命令
Alt + p 输入字符查找与字符相接近的历史命令
Alt + < 返回上一次执行命令

其它

快捷键 说明
Tab 补全命令、文件名和路径
Ctrl + s 锁住终端
Ctrl + q 解锁终端
Ctrl + l 清屏相当于命令clear
Ctrl + c 终止当前运行进程/另起一行
Ctrl + z 把当前进程送到后台处理
Ctrl + i 类似TAB健补全功能
Ctrl + o 重复执行命令
Ctrl + Alt + [Fn] 切换屏幕;从F1到F6是shell提示屏幕，F7是图形化屏幕。
Ctrl + Alt + Del 关机和重新引导，关闭当前的会话然后重新引导 ,只有在正常关机步骤不起作用时才使用这种方法
Alt + Tab 在图形化桌面环境中切换任务
Ctrl + x + u 撤销刚才的操作
鼠标选中字符串 选择字符串并且拷贝到剪贴板
鼠标中键 使用剪贴板的内容粘贴
